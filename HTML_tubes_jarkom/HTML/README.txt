Cara pakai:
1. Letakkan folder HTML ini sejajar dengan webserver.py.
2. Jalankan python webserver.py di laptop server.
3. Jalankan python proxy.py di laptop proxy.
4. Akses dari client:
   python client.py --mode tcp --path /index.html

Halaman tersedia:
- /index.html
- /tcpip.html
- /osi.html
- /qos.html
- /implementation.html
