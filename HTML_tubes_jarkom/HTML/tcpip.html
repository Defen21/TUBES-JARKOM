Model TCP/IP

TCP/IP adalah model komunikasi jaringan yang digunakan internet. Dalam tugas ini, layer aplikasi menggunakan HTTP sederhana, layer transport menggunakan TCP dan UDP, serta layer network menggunakan IP.
