Selamat Datang

Website ini digunakan sebagai aset pengujian untuk tugas besar Jaringan Komputer. Halaman ini disajikan oleh webserver.py, diteruskan oleh proxy.py, dan diminta oleh client.py.

Client

Mengirim HTTP request ke proxy dan menjalankan UDP ping untuk pengukuran QoS.

Proxy

Meneruskan request ke server, menyimpan cache, dan mengembalikan response ke client.

Web Server

Melayani file HTML statis melalui TCP dan memantulkan paket UDP sebagai echo server.

Alur Sistem

Client --TCP:8080--> Proxy --TCP:8000--> Web Server
Client --UDP:9000-----------------------> UDP Echo Server

Request HTTP wajib melewati proxy. Cache HIT berarti proxy mengirim response dari cache, sedangkan cache MISS berarti proxy meneruskan request ke web server.
