Implementasi Sistem

Sistem terdiri dari tiga file Python: client.py, proxy.py, dan webserver.py. Semua komunikasi jaringan menggunakan modul socket secara manual.

Konfigurasi Port

KomponenPortFungsi
Proxy8080Menerima HTTP request dari client
Web Server TCP8000Melayani file HTML
Web Server UDP9000UDP echo untuk QoS

Pengujian

python webserver.py
python proxy.py
python client.py --mode tcp --path /index.html
python client.py --mode udp
python client.py --mode multi --clients 5
