Quality of Service

QoS digunakan untuk mengukur kualitas komunikasi jaringan. Dalam tugas ini, parameter yang dihitung meliputi RTT, packet loss, jitter, dan throughput.

Parameter QoS

ParameterPenjelasan
RTTWaktu bolak-balik paket dari client ke server dan kembali ke client.
Packet LossPersentase paket UDP yang tidak mendapat balasan.
JitterVariasi delay antar paket.
ThroughputJumlah data berhasil diterima per satuan waktu.

Contoh Output UDP

Reply from 192.168.0.103: seq=1 RTT=2.15 ms
Reply from 192.168.0.103: seq=2 RTT=1.98 ms
Packet Loss: 0.00%
Avg RTT: 2.06 ms
